flexmark-java extension for HTML to Markdown extensible conversion

Extensible HTML to Markdown converter, allows inclusion of all custom extensions and creating your own:

Supported Extensions:

  • abbreviations
  • aside
  • block quotes
  • bold, italic, inline code
  • bullet and numbered lists
  • definition
  • fenced code
  • strike through
  • subscript
  • superscript
  • tables
  • multi-line URL images
